Employee Management System

HOW TO START

- pnpm i
- pnpm run dev

TECH STACKS

- PNPM
- NEXTJS
- TYPESCRIPT
- REDUX/TOOLKIT
- react-redux
- TAILWIND (headless ui, prelinescript)

FUNCTIONALITIES

- ADD RECORD
- EDIT RECORD
- DELETE RECORD
- VIEW RECORD
- SEARCH RECORDS
- SORTING RECORDS
- FORM VALIDATIONS (every field is required, with same id you can't add other employee, )
- PAGINATION
- Debouncing(for search)
- Responsive
